## Command Aliases
Some commands have aliases, they are included in the conf export at the end of every command
When entere, they should look like this;
**Aliases:**

```js
exports.conf = {
    DM: true,
    OwnerOnly: false,
    alias: ['alias1', 'alias2', 'alias3']
}
```
